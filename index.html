Week 1

Opdracht

Ga op Google en Github op zoek naar JavaScript libraries. Zoek daar er minimaal 10 van.

Schrijf in eigen woorden kort (ongeveer 3 zinnen). op wat ze doen.

1. PixiJS

Met PixiJS kan je super gave 2D animaties maken. Hiermee kan je bijvoorbeeld van pagina wisselen met een super gaaf smoke effect om de transactie van pagina naar een andere pagina te weergeven. PixiJS is snel, makkelijk te gebruiken en flexibel, je wordt dus niet snel beperkt in de mogelijkheden van PixiJS

2. D3.js

Wordt gebruikt om data te visualiseren. Het gooit een database-bestand om naar levendige charts, diagrammen, etc. Je kan deze gemaakte charts aanpassen door middel van CSS. Je kan zelf kiezen hoe je data eruit komt te zien door verschillende functies en design mogelijkheden.

3. React

Met ondersteuning van Facebook is React een van de grootste libraries. Het is een must voor front-end development. Je kunt met React creatieve web-apps ontwikkelen. Door de eindloze mogelijkheden van React verbeter je de klantenervaring van de bezoekers van je website.

4. Qunit

Maakt testen van je clients makkelijker, sneller en leuker. Met Qunit kan je javascript code sneller testen. Wat normaal een enorm veel tijd in beslag neemt. Qunit maakt dit makkelijker en sneller. Ook heeft het een grote community, waar je makkelijk je vragen kunt stellen als je bij een onderdeel vast loopt.

5. Parsley

Maakt goede overzichtlijke forms, en controleert ook of ze zijn ingevuld. Bijvoorbeeld of een wachtwoord is ingevuld, en of het wachtwoord zich ook houdt aan speciale cijfers, hoofdletters, etc. Parsley heeft ook veel focus op makkelijk gebruik voor de gebruikers. Het is eenvoudig en overzichtelijk.

6. jQuery UI

Zorgt ervoor dat je eenvoudig grafische elementen kunt toevoegen aan je website. Al deze elementen zijn te gebruiken op elke browser die er bestaat. Je kan pas gebruik maken van jQuery UI als je jQuery ook hebt geinstalleerd voor je pagina.

7. Chart.js

Met deze library kan je letterlijk alle soorten diagrammen maken van saaie data. Alles kan je designen door middel van CSS. Hiermee visualiseer je data, n.a.v. hiervan krijg je goede inzicht in je data. I.p.v. saaie lines code krijg je nu een overzichtelijk beeld van je data.

8. Anime.js

Hiermee kan je letterlijke alles animeren. Elementen laten veranderen van plek, en ze over een veld laten bewegen zijn allemaal mogelijk. Te gebruiken bij bijna alle grote browsers. Daarnaast is Anime.js snel, dus geen lags

9. Howler.js

Indien je van plan bent om geluid te implementeren op je website, is Howler een goede library om te gebruiken. Je kan je eigen music player maken, geluid afspelen in een emulator, of nummers uit elkaar halen. Ook voor game development kan je geluiden toevoegen wat het een extra gave feature maakt voor gamen.

10. Reveal.js

Perfect om powerpoint presentaties te maken. Je kan wisselen van slide, inzoomen, fades invoegen. Alles om een mooie overzichtelijke presentatie te maken. Reveal.js zorgt ervoor dat dit mogelijk is.
